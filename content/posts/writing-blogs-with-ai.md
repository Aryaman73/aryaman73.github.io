writing in the age of AI feels... strange. even as I type this, GitHub copilot is suggesting entire sentences for me to write. I can see the appeal of using AI to help with writing, but it also feels like I'm losing a part of myself in the process. the previous sentence was one of those suggestions.

on one hand, technical writing has always been about getting your point across. if AI can do a better job of that (quicker), then there's nothing wrong with using it. but it does me feel like a bit of a fraud...

i think the internet is going to find it's own footing somehow - i've noticed that lowercase has been a way for humans to be like "hey! i'm real!". but it is trivially easy for AI to copy these trends in a never ending game of whack-a-mole. github copilot finally caught on, and is now suggesting lowercase sentences for me to write.

to slop or not to slop, that is the question.
